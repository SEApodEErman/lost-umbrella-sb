# Asahina Akane - Lost Umbrella Storyboard
Storyboard for Lost Umbrella

Map [link.](https://osu.ppy.sh/beatmapsets/1740709#osu/3558572)

## Description
Music Video Recreation in the form of an osu! Storyboard.
This repo only contains the code used in storybrew to make the storyboard.

## Applications Used
[Storybrew](https://github.com/Damnae/storybrew) for programming the storyboard.
[Adobe Photoshop](https://www.adobe.com/sea/products/photoshop.html) For image manipulation.
[FFmpeg](https://ffmpeg.org/) for frame extraction from the MV.
[AegisSub](https://aegisub.en.uptodown.com/windows) for making the subtitle file.
